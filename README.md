# 🧠 Zyzz

> Your personal AI Chief of Staff.

---

## 🚀 Vision

Zyzz is a personal AI assistant that helps organize, think, plan and execute tasks using multiple AI models.

Instead of replacing the user, Zyzz acts as a partner capable of routing requests to the best AI, remembering context and automating repetitive work.

---

## 🎯 Mission

Build an AI assistant that genuinely saves time every day.
